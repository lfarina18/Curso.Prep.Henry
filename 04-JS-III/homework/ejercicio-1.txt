	* Arrays

    Un array es una variable que puede contener multiples elementos.
    Sirven para agrupar elementos del mismo tipo. El contenido va entre [], y cada elemento del array va separado por una coma.

    Un array puede tener multiples elementos que sean iguales

    Ejemplo:

    const numeros = [10, 20, 30];

    const meses = new Array("Enero", "Febrero", "Marzo");

    O un arreglo que contiene datos de todo tipo, ejemplo:

const deTodo = [
  "Hola",
  10,
  true,
  "si",
  null,
  { nombre: "Leo", trabajo: "Programador" },
  [1, 2, 3],
];

En definitiva:
Un array es una variable que puede contener multiples elementos
